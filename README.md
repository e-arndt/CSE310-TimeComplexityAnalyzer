# Time Complexity Analyzer

Module 1 – TypeScript Project

## Description
This is a command-line (CLI) TypeScript application that demonstrates different time complexities by allowing the user to select an algorithm and dataset size. The program runs the selected algorithm and displays execution time in the terminal.

## Features
- O(1) Constant Time
- O(log n) Binary Search (recursive)
- O(n) Linear Search
- O(n log n) Sorting
- O(n^2) Nested Loop

## Requirements Demonstrated
- Recursion
- Classes
- Lists (arrays)
- Async functions
- Terminal output
- Exception handling (try/catch)
